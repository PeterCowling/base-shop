Context & notes
Why these 20? — pipeline methodology

All 250 candidates were scored across five dimensions (D, W, P, E, I). The 150 names with available .com domains were ranked by Score descending, then by I (ICP resonance) for ties, then alphabetically. The top 20 were presented first. This extended view shows the next 20 in the ranked sequence — they meet the same domain-availability gate but scored slightly lower or were displaced by tie-breaking.

The highest score in this set is 24 (Calmalina at rank 21), which would have appeared earlier but its primary domain calmalina.com was taken — only the variant calmalinaa.com is available. This is a strong candidate if the operator is comfortable with a double-vowel domain.

Variant domains (double-vowel) — what they mean

Several names in this set have a "variant" domain — e.g., calmalinaa.com (double-a) instead of calmalina.com. This occurred because the generation agent created both a primary and an alternative when the primary domain was taken. The brand name itself is the spoken version (without the extra letter); the domain simply adds a suffix to make the .com available.

Variant domains have trade-offs: they're harder to spell by ear, but they are real, available, registerable .com addresses. If the brand name has enough pull, customers navigate by search rather than memory — mitigating the double-vowel risk.

Affected names in this set: Calmalina, Aluvinova, Nivedore, Tessitela, Levovera, Solavora, Tavirova.

MDR flag guidance — Anchorina & Comforteva

Anchorina (rank 35): The root "anchor" carries a physical securing/fixation connotation. Under EU MDR, marketing copy for cochlear implant accessories must not suggest the product aids or improves device function. A tagline like "anchors your sound" would be non-compliant. The name itself is not a regulatory problem — only the copy is governed. If selected, copy must treat the headband purely as a fashion/comfort accessory.

Comforteva (rank 38): The root "comfort" is a common marketing claim that could be interpreted as a benefit for the device wearer (e.g., "makes wearing the processor more comfortable"). This is riskier than Anchorina because "comfort" is a direct product attribute claim. Requires stricter copy guardrails. Low risk if framed purely aesthetically ("the look of ease"), high risk if framed functionally.

Pattern guide

Pattern A — Morpheme combination: Two invented morphemes fused together (e.g., calma+lina → Calmalina). High distinctiveness, fully ownable.

Pattern B — Craft/English respelling: An English word with an Italian-style ending (e.g., comfort+eva → Comforteva). Accessible to English speakers; moderate distinctiveness.

Pattern C — Fragment blend: Two syllable fragments blended (e.g., bari+ora → Bariora). Feels Italian and invented.

Pattern D — Root + suffix: A root word combined with a suffix (e.g., temira+atelier → temiraatelier.com). Spoken name differs from domain string. Strong visual wordmark on the domain; the spoken/brand name registers first.
