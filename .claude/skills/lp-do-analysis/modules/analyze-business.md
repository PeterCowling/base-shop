# Analysis Module: Business Artifact Track

Use for `Execution-Track: business-artifact`.

## Objective

Produce a decision-grade delivery strategy recommendation that planning can operationalize without reopening channel, artifact, or validation choices.

## Required Outputs

- Explicit option set
- Evaluation criteria
- Chosen delivery approach
- Rejected approaches with reasons
- Planning handoff notes

## Comparison Heuristics

- Compare expected upside, evidence quality, execution burden, approval friction, and validation speed.
- Prefer the smallest approach that can falsify the key hypothesis cleanly.
- Separate "interesting later" alternatives from the recommendation; do not blur them into the current plan.
