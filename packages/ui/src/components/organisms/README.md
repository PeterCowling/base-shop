# Organisms

Composite layout sections composed of atoms and molecules.

Organisms must expose `width`, `height`, `padding` and `margin`
props to control their rendered dimensions.

Current components:

- `Header`
- `SideNav`
- `Footer`
- `Content`
- `StatsGrid`
- `DataTable`
  `ProductCard`
- `ProductCarousel`
- `ProductGrid`
- `ProductFeatures`
- `ProductVariantSelector`
