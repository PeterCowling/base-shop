1. **Start with real examples** - Use actual code from your repo
2. **Include common mistakes** - Document what NOT to do

4. **Keep it focused** - One clear task per skill

6. **Show before/after** - Demonstrate the improvement
7. **Test the skill** - Try it with Claude before saving
