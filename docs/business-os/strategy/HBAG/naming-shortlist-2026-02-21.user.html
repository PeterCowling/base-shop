Pattern guide ‚Äî what each generation method means

Pattern A ‚Äî Morpheme combination

Two invented Italian-phonetic morpheme stems combined into a single word. Produces longer names (9-12 letters) with internal rhythm. Examples: Buralelova (bural+elova), Davelelora (davel+elora).

Pattern B ‚Äî Craft-term respelling

Italian craft or finishing vocabulary (brunire = to burnish; cesellare = to chisel) respelled with an invented suffix that removes any descriptiveness while preserving the craft phonetics. Examples: Bruneno, Cesellelo.

Pattern C ‚Äî Fragment blend

Two fragments from different Italian phonetic families blended at a natural phoneme junction. Produces 6-8 letter names with strong vowel flow. Examples: Celiena (celi+ena), Feniavo (feni+avo).

Pattern D ‚Äî Root + suffix (spoken name ‚â† domain)

Short invented root combined with a descriptive suffix (casa, atelier, mano) to form the domain string, while the spoken brand name is the root alone. Enables a short, memorable spoken name while maintaining a distinctive domain. Examples: Bravia / braviacasa.com.

Pattern E ‚Äî Real Italian word, non-descriptive in class

An obscure or archaic Italian word that is not descriptive of the goods in Classes 14/18/25 ‚Äî giving it better distinctiveness than a common-language word. Ponzatura is the only Pattern E name in this shortlist.
