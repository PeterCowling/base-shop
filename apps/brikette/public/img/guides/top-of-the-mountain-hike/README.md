## Required Images

- **Filename**: `01-view-capri.jpg`
- **Description**: View looking across the peninsula toward Capri
- **Source**: Google Doc - "top towards capri.jpg"

### Santa Maria → summit navigation (in-guide)
- **Filename**: `02-santa-maria-to-summit.jpg`
- **Description**: Directional image for the final section from Santa Maria del Castello to the summit

## Image Guidelines
- Keep filenames numbered to preserve navigation order in content.
- Prefer high-resolution sources when available; compress without significant quality loss.
- Format: JPEG or WebP preferred.
