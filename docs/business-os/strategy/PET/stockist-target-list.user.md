# PET Stockist Target List

This is the standing source for future physical-retail or wholesale doors for PET. The current
PET plan explicitly keeps boutique and wholesale outreach out of scope before the checkpoint, so
the file is a ready capture surface rather than an active rollout list.

## Current Activation Status

- Current PET go-to-market remains online-first.
- Physical boutique and wholesale outreach are intentionally deferred pre-checkpoint.
- When the checkpoint opens a stockist lane, record named targets here before any live sell-in.

## Data To Capture Once Targeting Starts

- Target name and location
- Buyer type and merchandising environment
- Price architecture and comparable products
- Terms requested (starter order, consignment, reorder cadence)
- Display requirements and content/UGC potential
- Decision owner and next follow-up date
